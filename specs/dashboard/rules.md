# Regras de Negócio — Dashboard

- Todos os dados são somente leitura (calculados em tempo real)
- Faturamento considera apenas vendas — não inclui orçamentos
- "Mês atual" = mês/ano corrente no fuso America/Manaus
- Top clientes e top produtos são baseados no período filtrado
- Período padrão quando não informado: mês atual
- Ticket médio = faturamento total / quantidade de vendas no período
